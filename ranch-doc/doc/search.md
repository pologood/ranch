# 搜索已发布文档信息集

请求
- Service Key - ranch.doc.search
- URI - /doc/search

参数

|名称|类型|必须|说明|
|---|---|---|---|
|category|char(100)|否|类别。|
|words|char(36)|是|分类ID。|
|pageSize|int|否|每页显示记录数，默认20。|
|pageNum|int|否|当前显示页数。|

返回值
```json
{
    "count": "总记录数",
    "size": "每页显示记录数",
    "number": "当前显示页数",
    "list": [
        {
            "id": "ID值",
            "classifies": ["分类集"],
            "author": {},
            "category": "类别",
            "sort": "顺序",
            "subject": "标题",
            "image": "主图URI地址",
            "thumbnail": "缩略图URI地址",

        }
    ]
}
```
