# 查找

请求
- Service Key - ranch.editor.find
- URI - /editor/find

参数

|名称|类型|必须|说明|
|---|---|---|---|
|id|char(36)|是|ID值。|
|user|char(36)|否|用户ID，为空则使用当前用户。|
